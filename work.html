Project Beyond (Jaar 2 HBO)

Project Beyond was een team project, je koos als team een irritatie waarvoor je een digitaal interactieve oplossing voor ging bedenken, zonder scherm. Wij hadden als Design Challenge voor dit project: Hoe kun je met een interactief ontwerp huishoudens met een rolcontainer, met een loopafstand van meer dan 70 meter van het huis (daardoor niet in het zicht), in huis stimuleren om de kliko op te halen wanneer het geleegd is.

Javascript Game (Jaar 1 HBO)

Als eindopdracht moesten we een eigen javascript game maken, in dit vak heb ik de basis geleerd van javascript en het leren begrijpen en toepassen. Hier heb ik een eigen game gemaakt met de designs van Genshin Impact, het doel in dit spel is de puzzle compleet maken, en voor een puzzle stuk moet je karakter werken, maar om te werken heeft je character ook energie nodig.
